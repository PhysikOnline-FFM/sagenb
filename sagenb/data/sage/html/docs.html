POKAL: Einstieg und Funktionen

Was ist POKAL?

POKAL ist eine studentische Online-Plattform zum kollaborativen Arbeiten- und Lernen. Dir steht als Nutzer eine leistungsstrake Mathematikengine zur Verfügung, mit der Du sowohl numerisch als auch symbolisch rechnen kannst. Durch die zusätzlichen Kollaborationsfunktionen, die das gleichzeitige Arbeiten von mehreren Nutzern and einem Worksheet erlauben, kannst Du somit gemeinsam mit Kommilitonen Übungsblätter bearbeiten, ASCII Daten visualisieren oder einfach Programmieren lernen. Genauer gesagt lässt sich in jeder Zelle nicht nur SAGE Code sondern auch gewöhnliches Python ausführen. Die Einsatzmöglichkeiten von POKAL sind daher sehr vielfältig.

Weitere Hilfe

SAGE Hilfe/Tutorials POKAL verwendet für Berechnungen standardmäßig die OpenSource Mathematik-Engine SAGE. Einen sinnvollen Einstieg stellen somit die SAGE Einsteiger-Tutorials dar. Alternativ gibt es thematisch geordnete Tutorials und die allgemeine SAGE Hilfe.
SAGE Dokumentation durchsuchen Durchsuche die SAGE Dokumentation direkt in einem Worksheet indem du search_doc("Suchbegriff") in einer Zelle evaluierst. Jeder reguläre Ausdruck ist als Suchbegriff geeignet.
Objekthilfe Schreibe ein '?' hinter ein Objekt und evaluiere die Zelle. Beispiel: plot?
